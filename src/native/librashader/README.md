# Bundled librashader runtime

EmuDOS renders CRT / slang shaders through **librashader** by
[SnowflakePowered](https://github.com/SnowflakePowered/librashader). All credit for the
shader engine goes to that project and its contributors.

librashader publishes no Linux release binary, so `fetch.sh` pulls the version-pinned,
SHA-256-verified prebuilt from the durable Arch Linux Archive and drops it here as
`librashader.so`. The build (`EmuDOS.csproj` → `FetchAndCopyLibrashader`) and
`packaging/build-release.sh` run this automatically and bundle the `.so` next to the
EmuDOS binary, so end users never have to install a separate package.

- Pinned version: **0.11.2** (C ABI version 2)
- License: dual **MPL-2.0 / GPL-3.0** (see `LICENSE-MPL.md`, `LICENSE-GPL.md`) — both
  compatible with EmuDOS's GPLv3. Also recorded in the repo's `NOTICES.txt`.

`librashader.so` itself is **not** committed (it's regenerated by `fetch.sh`); only the
script and licenses are tracked.
